# wien-radbauprogramm
Analyser for Wien's Radbauprogramm

## Installation
```sh
git clone https://github.com/plepe/wien-bauprogramm
cd wien-bauprogramm
npm install
```

## Usage
Extract current bauprogramm as JSON:
```sh
node cli.js
```

## Create API documentation
```
npm run doc
```
Open doc/index.html
